Light it up

Role
Programmer

Light it up was made using the Unity Engine in under 10 weeks.

My responsibilities included development of basic platforming, puzzles and other gameplay features such as checkpoints.

I also handled the majority of graphics implementation and animation key-framing.

Survive the darkness and bring light back into the world. Navigate through the darkest of caves, and retrieve a light crystal in order to repair the lighthouse.

Rubio

Role
Lead programmer

Rubio was made using the Unity Engine.

As lead programmer one of my responsibilities was to ensure that code meets technical requirements.

I worked on momentum-based platforming and basic combat systems using state-machines.

Fully modifiable platformer jump

The purpose of this project was to create a jumping script that can be reused in many different platforming games by simply creating a serialized jump-settings-file that affect the jumps look and feel.

An ongoing android game project in which you play as a gun. Made using Android Studio.
